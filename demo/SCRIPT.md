# 데모 영상 대본 (≈3분)

녹화 전 준비:
- 풀 런이 끝나 `results/scores.csv`, `results/fig1..5.png`, `results/generations.json`이 있어야 함.
- 터미널 폰트 크게, 창 넓게. 화면 녹화(Mac QuickTime / Win Game Bar·OBS) → mp4.
- `--slow`로 텍스트가 한 줄씩 떠서 카메라에 잘 잡힘.

> 팁: 오래 걸리는 `generate`/`evaluate`를 **라이브로 보여주지 않습니다.** 대신
> `showcase.py`로 "에이전트 1개 뉴스레터 처리 과정"을 ~30초에 보여주고(느낌 전달),
> 정량 결과는 미리 돌려둔 산출물을 띄웁니다.

## 0. 인트로 (20초) — 말로만

> "경량 오픈소스 SLM(Llama-3.2-3B)에 **역할을 분담**시키면, 대형 LLM의
> 뉴스레터 품질을 얼마나 따라잡을 수 있을까요? 그리고 비용은 얼마나 들까요?
> 4개의 에이전트 — Scout가 모으고, Reader가 요약, Writer가 작성,
> Editor가 평가하고 약하면 다시 피드백을 보냅니다."

(README의 파이프라인 표 한 장을 띄워두면 좋음)

## 1. 라이브 워크스루 (60초)

명령:
```bash
python demo/showcase.py --slow
```

화면을 따라가며 내레이션:

> "먼저 **Scout**가 HackerNews·arXiv에서 모아온 원본 기사 5개입니다."
> "**Reader**가 각 기사를 3줄로 충실히 요약합니다 — 한 번의 배치 호출로
> 5개를 병렬 처리하는 게 속도의 핵심이고요."
> "**Writer**가 이 요약들을 하나의 뉴스레터로 융합합니다."
> "여기서부터가 이 프로젝트의 핵심입니다. **Editor**가 초안을
> 사실성·일관성·가독성·간결성 4축으로 1~5점 채점하고(judge와 같은 축),
> 약하면 **한 줄 피드백**을 보냅니다."
> "기준 점수 미달이면 **Writer가 그 피드백을 반영해 다시 씁니다.**
> 이 자기수정 루프가 우리가 검증한 방법이고, 효과는 뒤에서 수치로 따집니다."
> "마지막에 같은 입력으로 **단일 SLM(한 방 생성)** 결과도 보여줍니다 —
> 분해 없이 통째로 쓰면 이렇게 나옵니다. 그 차이를 다음에 수치로 봅니다."

## 2. 정량 결과 (60초)

명령:
```bash
python demo/showcase.py --results
```

> "도메인당 15개, 총 30개 뉴스레터를 **동일한 고정 입력**으로 4개 시스템에 돌린
> 결과입니다."
> "**api**가 대형 LLM 상한선, **single_slm**이 하한선입니다."
> "핵심은 **faithfulness**(NLI 근거 일치율)입니다. **분해(multi_agent)** 가
> 단일 SLM의 25.6%를 32.7%로 끌어올려, 대형 LLM 상한선(31.2%)보다도 높습니다 —
> 기사별 요약(Reader)이 환각을 줄인 거예요."
> "그런데 여기에 **자기수정 루프(multi_agent_fb)** 를 더하면 오히려 충실도가
> 25.9%로 떨어지고 latency만 2.4배가 됩니다. 3B 모델이 자기 비평을 제대로
> 반영하지 못하기 때문입니다."
> "judge 점수 차이는 작아서 보조로만 보고, 충실도와 win-rate로 읽는 게 정확합니다."

(표 아래로 시스템별 샘플 뉴스레터가 같이 나옴 — 한두 개 읽어주기)

## 3. 그림 (40초)

```bash
# results/fig1..5.png 를 이미지 뷰어로 열기
```

> "슬라이드용 그림입니다. fig4(충실도)가 핵심 — 분해가 충실도를 ceiling 위로

## 4. 마무리 (20초)

> "정리하면 두 가지입니다. 첫째, **역할 분해는 효과가 있습니다** — 작은 모델로도
> 대형 LLM을 넘는 충실도를 냅니다. 둘째, **단순 자기수정은 오히려 해롭습니다** —
> 3B 리바이저가 비평을 소화 못 하기 때문이고, 이게 핵심 발견입니다:
> **self-correction의 효용은 베이스 모델 역량에 달려 있다.**
> 데이터는 전부 공식 무료 API, 평가셋은 고정해 재현 가능합니다. 감사합니다."
